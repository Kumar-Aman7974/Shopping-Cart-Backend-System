echo "# Shopping Cart Backend System

## 🛒 E-Commerce Backend API

A complete Spring Boot backend system for an e-commerce platform with shopping cart functionality.

## 🚀 Features
- User management
- Product management with categories
- Shopping cart operations
- Order processing
- Image upload support

## 🛠️ Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- Maven
- MySQL/PostgreSQL

## 📦 Installation

\`\`\`bash
git clone https://github.com/Kumar-Aman7974/Shopping-Cart-Backend-System.git
cd Shopping-Cart-Backend-System
mvn spring-boot:run
\`\`\`

## 🔧 Configuration
Update \`application.properties\` with your database credentials.

## 📝 API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | /api/products | Get all products |
| POST | /api/cart | Add to cart |
| POST | /api/orders | Place order |
